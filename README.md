Klassenbuch

Ein kleines Java-Projekt zur Verwaltung eines Klassenbuchs.
Das Programm speichert Informationen über Schüler, Klassen und schulische Daten.

Funktionen
Schüler verwalten
Klassen verwalten
Informationen anzeigen
Arbeiten mit Listen und Objekten
Konsolenausgaben
Projektstruktur

Das Projekt besteht aus mehreren Klassen, zum Beispiel:

Schueler
Klasse
Klassenbuch
Main

Jede Klasse übernimmt eine eigene Aufgabe im Programm.
